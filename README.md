**QR-Convert** is a premium, high-precision web-based tool designed to convert Static QRIS (Quick Response Code Indonesian Standard) into Dynamic ones instantly. Built with a focus on security, elite aesthetics, and a seamless user experience.

## âœ¨ Features

- **Modern Obsidian & Emerald UI**: A sleek, luxury dark-themed interface designed for a professional look.
- **Dynamic Conversion**: Effortlessly add specific transaction amounts to your static QRIS.
- **Smart Service Fees**: Includes options to add fixed (IDR) or percentage-based service fees.
- **Quick Nominal Chips**: Instant selection buttons for common amounts like 10k, 20k, 50k, and 100k.
- **Professional Animations**: Features a smooth "Circle-to-QR" morphing transition upon generation.
- **HD Download**: Save your generated dynamic QRIS as a high-quality PNG image.
- **100% Offline & Secure**: All data processing happens locally in your browser, ensuring total privacy.

## ðŸ›  Technical Setup

### Core Technologies
- **HTML5 & CSS3**: Custom Bricolage Grotesque typography and responsive obsidian layout.
- **JavaScript (ES6+)**: Core logic for QRIS string manipulation and CRC16 CCITT (0x1021) calculation.
- **jsQR**: Robust library used for decoding uploaded static QRIS images.
- **QRCode.js**: Library used for generating high-definition dynamic QR codes.

### How It Works
1. **Upload**: Select your static QRIS image.
2. **Input**: Enter the transaction amount and set any optional service fees.
3. **Generate**: Click the button to watch the professional animation as your QR is created.
4. **Result**: Scan the QR directly or download the HD image for later use.
